# Бизнес-анализ: требования, процессы, риски (кейс: [Кофейня] )

## Раздел 1. Требования 
- **Бриф** (шаблон + заполненный): цели, проблемы, KPI, ограничения.
- **Проблемное поле**: ключевая причина — падение лояльности / рост дефектов / снижение чека.
- **Оценка инициатив**: выбрана программа лояльности (NPV +, окупаемость 4 мес., применимость высокая).
- **Сценарий сессии**: ЛПР — управляющий, роли: фасилитатор, владелец процесса, маркетолог, IT, финансист.
- **План по Исикаве**: 4 мероприятия (опросы, обучение, автоматизация, контроль).
- **Стратегическая карта**: финансы → клиенты → процессы → персонал.

## Раздел 2. Процессы 
- **Реестр процессов** (по ELMA): Уровень 0–2.
- **Диаграмма EPC** (нотация): ключевой процесс «Обслуживание / Производство / Отгрузка».
- **Визуальный анализ**: найдены разрывы, дублирование операций, узкие места.

## Раздел 3. Влияние и риски 
- **Матрица стейкхолдеров**: ключевой — **персонал** (власть/интерес высокий). Стратегия: вовлечение + мотивация.
- **Риски**: топ-3 — текучесть, сбои поставок, падение спроса.
- **Анализ рисков**: ключевой — **текучесть** (вероятность 0.7, ущерб высокий).
- **План по риску**: наставничество, KPI для удержания, exit-интервью.
- **Мониторинг**: коррекция (доработка графика), корректирующее (замена сотрудника), предупреждающее (обучение).
- **Эффект от процессного подхода**: снижение потерь на 12%, экономия ≈ 500 тыс. руб./год.
